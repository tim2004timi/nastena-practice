# Создание иконки приложения

Для создания иконки приложения с шапочкой выпускника:

## Быстрый способ

1. Откройте любой графический редактор (Paint, GIMP, Photoshop, или онлайн редактор)
2. Создайте изображение 1024x1024 пикселей
3. Нарисуйте или вставьте иконку шапочки выпускника
4. Используйте фиолетовый фон (#9D4EDD) как на странице входа
5. Сохраните как `assets/icon/app_icon.png`
6. Создайте версию без фона (только иконка) как `assets/icon/app_icon_foreground.png`

## Онлайн инструменты

- https://www.fluttericon.com/ - генератор иконок Flutter
- https://appicon.co/ - генератор иконок приложений
- https://icon.kitchen/ - генератор адаптивных иконок

## После создания файлов

Запустите:
```bash
flutter pub run flutter_launcher_icons
```

Это создаст все необходимые размеры иконок для Android.
