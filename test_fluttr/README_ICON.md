# Инструкция по созданию иконки приложения

Для создания иконки приложения с иконкой GraduationCap (как на странице входа):

## Вариант 1: Использовать онлайн инструмент (рекомендуется)

1. Перейдите на https://www.fluttericon.com/ или https://appicon.co/
2. Загрузите или создайте иконку с шапочкой выпускника (GraduationCap)
3. Скачайте созданные иконки
4. Сохраните основную иконку как `assets/icon/app_icon.png` (1024x1024)
5. Сохраните foreground версию (только иконка без фона) как `assets/icon/app_icon_foreground.png` (1024x1024)

## Вариант 2: Создать вручную

1. Создайте PNG файл 1024x1024 пикселей
2. Нарисуйте иконку шапочки выпускника на фиолетовом фоне (#9D4EDD)
3. Сохраните как `assets/icon/app_icon.png`
4. Создайте версию без фона (только иконка) как `assets/icon/app_icon_foreground.png`

## После создания иконок

Запустите команду для генерации всех размеров иконок:
```bash
flutter pub get
flutter pub run flutter_launcher_icons
```

Иконки будут автоматически созданы для всех необходимых размеров Android.
